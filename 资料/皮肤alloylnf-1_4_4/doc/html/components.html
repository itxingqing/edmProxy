Customization of Components

This page list possible customizations for single components. The list will be extended in future releases.
Currently covered components:
  • JButton/JToggleButton
  • JList
  • JPopupMenu
  • JTextField/JTextArea

Setting a custom background color
There are two ways for setting a background color of a button.

1. To have a flat, monochrome background use the method setBackground(Color bg).
2. In order to set a custom background color with a 3D effect, use the static method set3DBackground(AbstractButton b, Color c) of the class com.incors.plaf.alloy.AlloyCommonUtilities.

Disabling the pulse effect on default buttons
You can disable the pulse effect on default buttons by setting the property "alloy.isButtonPulseEffectEnabled" to "false". Look at the section about properties for details on usage.


Setting a custom border
The Alloy Look&Feel provides some custom borders for buttons. These borders are designed for special cases, in which the standard border is not the best solution. The the borders are created by static methods of the class com.incors.plaf.alloy.AlloyCommonBorderFactory.
  • createSmallButtonBorder(): Creates a border which consumes less space than the standard border. It can be used for small tool bar buttons
  • createAlternativeToolBarButtonBorder(): Creates a border that is always painted when the border is located in a toolbar. This can improve visibility of the button, especially for buttons that display text.

JList

Making the background tranlucent
To make the background of a list translucent you have to set the text field to non-opaque (using setOpaque(false) ). Then set the background color to a translucent color (example: setBackground(new Color(255, 255, 255, 128)) ). Additionally you may have to set the containing viewport and scrollpane to non-opaque.

JPopupMenu

Disabling the shadow effect
You can disable the shadow effects on popup menus (and on the popup of combo boxes) by setting the property "alloy.isPopupShadowEffectEnabled" to "false". Look at the section about properties for details on usage.

JTextField/JTextArea

Setting the non-editable background
There are several ways to set the non-editable background
  1. Use the method setBackground(Color bg) in class JTextField. This will set the editable and non-editable background to the specified color.
  2. Set the UI-default "TextField.lockedBackground" to specify the non-editable background for all text fields.
  3. Set the client property "lockedBackground". You can set a client property with the method putClientProperty() of the class JTextField.

Making the background translucent
To make the background of a text field translucent you have to set the text field to non-opaque (using setOpaque(false) ). Then set the background color to a translucent color (example: setBackground(new Color(255, 255, 255, 128)) ).
